# Project Overview

## Purpose
ReportBuilder is a Next.js 15 web application project, currently in its initial state as created by `create-next-app`. The project appears to be designed for building reports, though specific functionality is not yet implemented.

## Tech Stack
- **Framework**: Next.js 15.4.6 (App Router)
- **Frontend**: React 19.1.0
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS v4 with PostCSS
- **Fonts**: Geist Sans and Geist Mono from next/font/google
- **Linting**: ESLint with Next.js TypeScript configuration
- **Development**: Node.js with npm package management

## Current State
This is a fresh Next.js project with the standard template structure. No custom business logic has been implemented yet - it contains only the default Next.js starter template with Tailwind CSS integration.
