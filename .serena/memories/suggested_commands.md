# Development Commands

## Primary Development Commands

### Development Server
```bash
npm run dev
```
Starts the Next.js development server on http://localhost:3000 with hot reload

### Production Build
```bash
npm run build
```
Creates an optimized production build of the application

### Production Server
```bash
npm start
```
Starts the production server (requires `npm run build` first)

### Linting
```bash
npm run lint
```
Runs ESLint with Next.js TypeScript configuration to check code quality

## Alternative Package Manager Commands
The project supports multiple package managers:
- npm: `npm run dev`, `npm run build`, etc.
- yarn: `yarn dev`, `yarn build`, etc.
- pnpm: `pnpm dev`, `pnpm build`, etc.
- bun: `bun dev`, `bun build`, etc.

## Git Commands
Standard git workflow commands:
- `git add .` - Stage all changes
- `git commit -m "message"` - Commit changes
- `git push` - Push to remote repository
- `git pull` - Pull latest changes

## System Commands (macOS/Darwin)
- `ls` - List directory contents
- `cd` - Change directory
- `grep` - Search text patterns
- `find` - Find files and directories
